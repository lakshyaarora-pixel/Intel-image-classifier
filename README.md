# 🌍 Intel Image Classification App

A deep learning project that classifies natural scenes using CNN + MobileNetV2 and provides predictions via a Streamlit web app.

## 🚀 Features
- Image classification (6 classes)
- Transfer Learning (MobileNetV2)
- Streamlit UI
- Top predictions + confidence

## 🧠 Classes
- Buildings
- Forest
- Glacier
- Mountain
- Sea
- Street

## 🛠️ Tech Stack
- TensorFlow / Keras
- Streamlit
- NumPy
- PIL

## 📊 Model Performance
- Training Accuracy: ~90%
- Validation Accuracy: ~85-90%

## ▶️ Run Locally

```bash
pip install -r requirements.txt
streamlit run app.py
